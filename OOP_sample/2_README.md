### 음악 플레이어 만들기

## Song
- 각 곡을 나타내는 클래스
- 작곡가, 곡 이름, 플레이 시간(ms)을 상태로 가짐
- play 메서드는 Song 개체의 정보를 화면에 출력

## Playlist
- 재생목록 기능. 즐겨 듣는 음악을 추가하고 이름을 지정할 수 있음
- 플레이 리스트 이름, 곡들을 상태로 가짐
- 곡 추가, 제거 메서드를 가짐

## JkTunes
- 곡과 플레이리스트들을 상태로 가짐
- 곡 개수를 카운트하는 메서드, 곡 추가/제거, play하는 메서드를 가짐
- 플레이 리스트를 추가/제거, play하는 메서드를 가짐
